# Aegis

HUD with vital signs, combo points, and incoming pressure tracking

Tested on Project Ascension (WoW WoTLK 3.3.5a).

## Install

Download the latest release zip from the [Releases page](https://github.com/spyspott3d/Aegis/releases).
Extract it into your `World of Warcraft/Interface/AddOns/` folder so the path looks like `Interface/AddOns/Aegis/Aegis.toc`.
Launch the client.

At login, type `/ae about` in chat to confirm the addon is loaded.

## Commands

`/ae help` lists all available commands.

`/ae about` shows version info.

## Bug reports

Open an issue with the addon version (visible via `/ae about`), the realm and client build, and steps to reproduce.
